# Desarrollo de Funcionalidades

## Parámetros
- **$ARGUMENTS**: Descripción del feature o path a un documento funcional (.md)

## Ejemplos
```
/dev Agregar selector de seguro de vida opcional al simulador
/dev docs/feature-comparador.md
/dev Cambiar el cálculo de TEA para incluir gastos administrativos
```
